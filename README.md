# Naman's Story — Fun Hire Me Page

A playful, interactive static website that tells Naman's job search story.

## How it works

1. Open `index.html` in any browser (double-click or drag into Chrome/Edge/Firefox).
2. Click the big **"Click Me"** button — each click reveals more of the story.
3. After 3 clicks, the full story + hire CTA appears with confetti.

## Customize before sharing

Edit `index.html` and update:

- **Email** — `naman20jain04@gmail.com` (Hire Me button opens a pre-filled professional email)
- **LinkedIn** — replace `https://linkedin.com` with your profile URL
- **GitHub** — replace `https://github.com` with your profile URL
- **Skills** — add/remove tags in the skills section
- **Name** — update the `<h1>` and footer if needed

## Deploy for free (optional)

- **GitHub Pages**: push repo → Settings → Pages → deploy from `main` branch
- **Netlify**: drag the `DODO` folder to [netlify.com/drop](https://app.netlify.com/drop)
- **Vercel**: `npx vercel` in this folder

No build step needed — it's pure HTML/CSS/JS.
